# 微调与对齐

> AI Agent 面试题 - 模型微调与对齐技术

---

## 模块目标

本模块覆盖大模型的微调与对齐技术，包括 SFT、RLHF、高效微调方法等。

面试中常被问到：
- "SFT 和 RLHF 有什么区别？"
- "LoRA 的原理是什么？为什么能节省显存？"
- "如何构建高质量的指令微调数据？"

---

## 待补充知识点

### P2 - 一般

- [ ] SFT 监督微调
  - 全参数微调
  - 微调数据构建
  - 过拟合问题
  - 学习率调度

- [ ] RLHF 原理
  - 奖励模型训练
  - PPO 算法
  - DPO (Direct Preference Optimization)
  - RLHF 的局限性

- [ ] LoRA/QLoRA 高效微调
  - 低秩适配原理
  - 可训练参数比例
  - 显存节省计算
  - QLoRA 的量化策略

- [ ] 指令微调数据构建
  - 数据多样性
  - 指令模板设计
  - 数据质量过滤
  - 数据配比
