# 多模态 Agent

> AI Agent 面试题 - 多模态理解与交互

---

## 模块目标

本模块覆盖多模态 Agent 的基础知识，包括视觉语言模型、图文检索等。

面试中常被问到：
- "多模态 Agent 的架构是怎样的？"
- "VLM 是如何处理图像输入的？"
- "Agent 如何调用图像处理工具？"

---

## 待补充知识点

  - 图像理解工具
