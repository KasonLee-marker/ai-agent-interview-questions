# AI Agent 开发框架

## 主流框架对比

| 框架 | 语言 | 特点 | 适用场景 |
|------|------|------|----------|
| **LangChain** | Python/JS | 生态最丰富，组件化设计 | 通用 Agent 开发 |
| **LlamaIndex** | Python | 专注 RAG，数据检索强 | 知识库问答 |
| **AutoGPT** | Python | 全自动执行，目标驱动 | 自主任务执行 |
| **CrewAI** | Python | 多 Agent 协作框架 | 团队协作场景 |
| **Semantic Kernel** | C#/Python/Java | 微软出品，企业级 | 企业应用 |
| **Spring AI** | Java | Spring 生态集成 | Java 企业开发 |
| **LangChain4j** | Java | LangChain Java 移植 | Java 开发者 |
| **OpenClaw** | 多语言 | 你正在用的 😄 | 个人自动化 |

## 框架选择建议

### Python 开发者
- **快速原型**: LangChain
- **RAG 应用**: LlamaIndex
- **多 Agent**: CrewAI

### Java 开发者
- **Spring 项目**: Spring AI
- **LangChain 生态**: LangChain4j

### 企业级应用
- **微软生态**: Semantic Kernel
- **Spring 生态**: Spring AI

## 面试题

1. **LangChain 和 LlamaIndex 的核心区别？**
   - LangChain 是通用 Agent 框架
   - LlamaIndex 专注数据索引和检索

2. **如何选择合适的 Agent 框架？**
   - 技术栈匹配
   - 社区活跃度
   - 生产就绪程度
   - 性能要求
